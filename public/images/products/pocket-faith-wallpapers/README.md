# Pocket Faith Wallpapers — preview images

Drop preview JPGs of the actual wallpapers here, named **1.jpg through 8.jpg**.

The product page automatically uses them in two places:

- **Hero** — picks up `1.jpg`, `2.jpg`, `3.jpg` for the three floating phone mockups
- **"Inside the pack" gallery** — shows all 8 (`1.jpg` through `8.jpg`) in a grid

If a file is missing, that slot quietly hides itself and the gradient placeholder
is used instead — no errors, page still works.

## Recommended specs

- Format: JPG (or WebP if you prefer)
- Aspect: 9:19 (phone lockscreen) — anything close works, they're cropped to fit
- Width: 800–1200 px is plenty for previews. Keep filesize < 200 KB each.
- Use https://tinypng.com to compress before committing.

These are **previews**, not the actual high-res files buyers download. Those go
into the ZIP you upload via the Stripe redirect flow.
